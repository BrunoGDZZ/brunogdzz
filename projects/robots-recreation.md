  <h1>Robots Recreation</h1>

  <p class="project-page-intro">
    A Java desktop recreation of the classic <em>Robots</em> game built with Java Swing.
    The project focuses on object-oriented design, clean game logic, and a structured UI.
  </p>

  <div class="project-page-section">
    <h2>Overview</h2>
    <p>
      This project recreates the gameplay of the classic Robots game, where the player
      must survive while robots move toward them each turn. I used Java and Swing to build
      the interface and organize the game into manageable classes.
    </p>
  </div>

  <div class="project-page-section">
    <h2>Technologies Used</h2>

      <li>Git/GitHub</li>
    </ul>
  </div>
